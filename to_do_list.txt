我们现在要做的是第三步，有了第二步backmap的结构之后，我们要把这个All atom结构在PACE（我们的分子力场）中建模。

任务分解如下： 

1. 生成和我们蛋白质结构序列相同的蛋白质的拓扑文件，应当从之前前几步的输入如yaml、fasta（IDP）、 pdb（MDP）里面读取序列，输入给prepare_peptide.py以生成拓扑文件，并修改拓扑文件最后的单体的数目与CG模拟匹配。

2. 调用pdb2gmx处理backmap之后的结构，我们使用pdb2gmx命令来处理成适配PACE的结构。这一步使用GROMACSWRAPPER完成，示例代码如下。
import gromacs

# The command-line flags become keyword arguments.
# The flag name (without the '-') is the argument name.
# Flags without a value (like -ignh) are set to True.
gromacs.pdb2gmx(
    f="backmapped_aa_final_ter.pdb",
    ff="pace-new",
    water="no",
    ignh=True
)

3. 使用OPENMM来优化我们的结构，具体请参考提供的文件。
